## 常用库

### 切片

js 通过切片模拟栈和队列

栈

```js
// 创建栈
let stack = []
// push压入
stack.push(10)
// pop弹出
stack.pop()
stack.length--
// 检查栈空
stack.length === 0
```

队列

```js
// 创建队列
let queue = []
// enqueue入队
queue.push(10)
// dequeue出队
queue.shift()
queue.slice(1)
// 长度0为空
queue.length === 0
```

注意点

- slice(start,end) 返回一个新的数组
- 默认 s=s[0:len(s)]，取下限不取上限，数学表示为：[)
